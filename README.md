# Supplementary Materials for "A Corpus-Based Model of Lexico-Syntactic Compensation..."

This repository contains the supplementary data for the paper:

> Malkova, N.V. (2023). A Corpus-Based Model of Lexico-Syntactic Compensation: Evaluating Systemic Divergence in Russian and English Rhetorical Strategies. *Language Resources and Evaluation*.

## Contents

*   **[discourse-markers.md](discourse-markers.md):** The complete lists of potential discourse markers (N=76 for each language) used for the automated search phase of the study.
*   **[editorial-guidelines-analysis.md](editorial-guidelines-analysis.md):** The results of the content analysis of author guidelines from 20 academic journals (10 Russian, 10 English) performed to control for the variable of editorial policy.
